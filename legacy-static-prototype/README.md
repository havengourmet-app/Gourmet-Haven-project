# Legacy Static Prototype

This folder contains the archived HTML, CSS, JavaScript, image, and video prototype that existed before Gourmet Haven moved to the React + Express workspace.

## Structure

```text
legacy-static-prototype/
|-- media/
|-- scripts/
|-- styles/
|-- index.html
|-- login.html
|-- signup.html
|-- user-homepage.html
|-- owner-homepage.html
|-- orders.html
`-- profile.html
```

## Usage

The active project is now `../gourmet-haven/`. These files are kept only for design reference and should not receive new product features.

If you want to view the old prototype, serve this folder with a simple static server and open `index.html`.
